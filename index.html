uint32_t mul_mod (uint32_t a, uint32_t b, uint32_t mod) {
// defines the variables needed by mul_mod
uint32_t result = 0;
b = b % mod;

while (a != 0) {
if (a % 2 == 1) {
result = (result + b) % mod;
}
a = a >> 1;
b = (b << 1) % mod;
}

return result;
}
This function determines the product of a and b and returns the modulus of that by the number mod. This function is designed for use on the Ardiuno Mega, where an unsigned long is represented by 32 bits.
uint32_t pow_mod (uint32_t base, uint32_t exponent, uint32_t mod) {
// defines the variables needed by pow_mod
uint32_t result = 1;
uint32_t power = base % mod;

while (exponent != 0) {
if ((exponent % 2) == 1) {
result = mul_mod(result, power, mod);
}
exponent = exponent >> 1;
power = mul_mod(power, power, mod);
}

return result;
}
This function determines the modulus of base to the power of exponent by mod. This function uses mul_mod to prevent overflow when multiplying large quantities. Designed for the Arduino Mega.

function pull_twitch_info(user){
var src = "https://api.twitch.tv/kraken/streams/" + user;

$.ajax({
url: src,
type: 'get',
dataType: 'jsonp',
success:function(twitch_info){
if (twitch_info.stream == null) {
// execute if not streaming
} else {
// execute if streaming
$("#live").css('display', 'inline-block');
}
}
});
}
This Java-script uses Twitch's API to check if a user is streaming. This function can be used to edit parts of a web-page to indicate if a user is streaming. It also can grab information about the current streaming session.
